Beleza pessoal? Bom gostaria de começar este post falando um pouco sobre a plataforma que adotei, algumas vantagens que vejo e o porque das melhorias

Plataforma

Utilizo Jekyll um gerador de sites super simples mais extremamente poderoso com alguns recursos que adoro como:

  1. Site estático.
  2. Javascript, CSS e HTML compilados.
  3. Integração com Disqus.
  4. Hospedado diretamente no Github pages.
  5. Deploy extremamente fácil.
  6. Personalização completa.

PS: Fora as vantagens que Github pages como domínio personalizado.

Novidades desta versão

  1. Design clenar
  2. Deploy inteligente
  3. Busca nos artigos (não implementado na versão anterior).
  4. Nuporte melhor a responsividade.
  5. Suporte melhor a highlight.

E fora as melhorias apresentadas houve uma reestruturação na arquitetura e o gerenciamente de tasks com rake.

Enfim as novidades estão apenas começando e muito artigo bom está para ser publicado em breve, aguardem.
